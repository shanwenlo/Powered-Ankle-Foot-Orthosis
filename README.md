# Powered Ankle-Foot Orthosis

PAFO_Integrated is the full integrated code for 30 seconds of operation. Prints out all sensor data in .csv file.

GaitTrial_IMU_only is used for gait trails without the device.

Testing and Inegration code is used for bench testing sensors

Gait_Metrics_Calculation_perTrial_V2.m is used to calculate RMS accleration and CoV from collected gait data. Atleast 20 strides are required. Add collected sensor data .csv files to a folder and change folder path in MATLAB code.
